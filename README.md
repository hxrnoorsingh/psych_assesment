# Internet-Induced Learning Difficulties Assessment Tool
## PDM-2 Framework for Adolescents

A scientifically grounded, psychodynamic self-assessment tool designed to evaluate the impact of internet usage on adolescent learning capabilities. Built on the **Psychodynamic Diagnostic Manual (PDM-2)** framework.

## ÔøΩ Purpose

This tool investigates the correlation between excessive internet usage and maladaptive patterns in adolescents (ages 12-18), specifically focusing on:

- **Mental Functioning (MA Axis):** Attention, impulse control, and self-reflection.
- **Personality Patterns (PA Axis):** Maladaptive behaviors related to internet addiction.
- **Symptomatic Distress (SA Axis):** Anxiety, depression, and emotional dysregulation.

## ÔøΩ Assessment Structure

The assessment consists of **20 research-aligned questions** mapped to three axes:

1. **MA Axis (Mental Functioning)** - 7 items
2. **PA Axis (Personality Patterns)** - 7 items
3. **SA Axis (Symptomatic Distress)** - 6 items

Scoring is percentage-based (0-100%), with higher scores indicating greater difficulty or distress.
